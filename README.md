# gr-tools

Commands to make my life easier

## Requirements

* Node.js
* NPM
* Debian based OS

## Getting started

```sh
npm install -g gr-tools
gr-tools --version
gr-tools --help
```

### Scripts

#### Development

```sh
npm run dev
```

Removes possible `gr-tools` globally installed and executes `npm link` for testing while developing.
